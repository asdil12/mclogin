Setting up Client

  1. Download Weberver Certificate java.crt
  2. Import it: sudo keytool -import -alias login.minecraft.net -file java.crt -keystore $JRE_HOME/lib/security/cacerts -storepass changeit -noprompt
    (To remove it: sudo keytool -delete -alias login.minecraft.net -keystore $JRE_HOME/lib/security/cacerts -storepass changeit )
  3. Download minecraft_launcher.jar
  4. Run java -jar minecraft_launcher.jar

Setting up Server

  1. Download minecraft_server.jar
  2. Place it in a new directory
  3. Run java -jar minecraft_server.jar
